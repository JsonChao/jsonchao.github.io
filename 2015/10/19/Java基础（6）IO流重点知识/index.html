Java基础（6）IO流重点知识

Posté le 2015-10-19 | In 安卓基础 | Visitors:

Hello，这是java的第六堂课

1、IO流:(Input Output)处理设备之间的数据传输

字节流可以操作任何类型的数据：文本，音频，视频，图片
字符流：只能操作文本类型的数据，基于字节流的，在字节流的基础上融入了编码ASCII,ISO8859-1,gbk,gb2312,utf-8

2、设备：硬盘，键盘，控制台，内存，网络

最常用的设备是硬盘

数据没有写入到文件，写入到流内部的的一个数组，这个数组是一个字节数组，因为要查编码表

fw.write("abcdef");//把内部缓冲中的数据写入到文件
    fw.flush();
fw.write("gh");//刷完之后可以继续写
fw.close();//关闭此流，同时会刷新流中的数据
fw = new FileWriter("k:\\temp.txt");//因为该路径不存在，所以文件输出流对象创建失败

5、字符流的缓冲区：只是为了提高读或写的效率，不具备读或写的功能，不能单独使用，需要结合着读流或写流使用

字符流的缓冲读取流：
BufferedReader，提供了一个一次读取一行的功能 readLine()

字符流的缓冲写入流: BufferedWriter:提供了一个跨平台的写入换行的功能 newLine()

bw.close();//关闭缓冲流就是关闭写入流

12、当遇到一个IO问题是，该怎么考虑?

1:是否有源和目的
  有源:Reader  InputStream
  有目的: Writer OutputStream
2:是不是文本
  是文本：   Reader Writer
  不是文本: InputStream OutputStream
3:根据第二部确定的体系，进一步的确定操作的是哪个设备
  源：键盘，硬盘，内存，网络
  目的：控制台，硬盘，内存，网络

20、RandomAccessFile:只能访问文件,不属于IO体系,内部既有字节输入流，也有字节输出流,内部有一个字节数组，可以通过指针操作该数组，所以可以随机

在"rw"模式下，文件不存在会自动创建 
RandomAccessFile  random = new RandomAccessFile("random.txt","rw");
//默认从文件开头写入
random.write("刘能".getBytes());
random.writeInt(56);
//获取指针指向的位置
long index = random.getFilePointer();
//设置指针的位置
random.seek(16);
