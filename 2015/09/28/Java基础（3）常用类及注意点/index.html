Java基础（3）常用类及注意点

Posted on 2015-09-28 | In 安卓基础

Hello，这是java的第三堂课

1、局部内部类:如果局部内部类使用了其所在函数的局部变量，那么该局部变量必须是final的

2、throws后边跟的是异常类，throw后边跟的是异常类对象

3、字符串是常量；它们的值在创建之后不能更改

4、String这个类重写了 Object类中的 boolean equals(Object obj),比较的是字符串内容

5、将字符串转成字节数组

byte[] getBytes()

6、切割: 

String[] split(String)//括号里的String为按照什么样的格式切割

String str = ss.replace('l','w');//返回一个新的字符串，原有的字符串没有改变

7、将基本数据类型转换成字符串

String.valueOf()//括号里面的为要转换的基本数据类型

8、jdk升级的原因：

1：简化书写
2：提高安全性
3：提高效率

9、

StringBuffer sb1 = sb.append(false);//添加完之后还返回原容器

10、其它进制转十进制

Integer.parseInt(String,int);

11、一个数在字节允许的范围内(-128—127)，如果之前已经定义过，那么再使用时就直接使用已有的

12、把Date类型的时间值转成long类型的

Date d = new Date();
long shijian = d.getTime();

13、得到一个DateFormat类型的实例

DateFormat  df = DateFormat.getDateTimeInstance();
