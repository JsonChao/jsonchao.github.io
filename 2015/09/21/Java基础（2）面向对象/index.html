Java基础（2）面向对象

Posted on 2015-09-21 | In 安卓基础

Hello，这是java的第二堂课

1、静态只能用静态，非静态什么都能用

2、构造代码块可以用于所有对象共性内容的初始化

3、构造函数只能用于每个对象特有内容的初始化

4、一个对象的初始化过程：

1：因为创建对象需要字节码文件，所以先加载字节码文件到方法区
2:如果有静态代码块执行静态代码块
3：在堆中开辟内存
4：为成员变量赋默认值
5：为成员变量赋初始值
6：执行构造代码块
7: 执行构造函数
8：把对象在堆中的内存地址赋给栈中的变量

5、继承中成员变量的特点

this是一个引用，指向当前对象

super:不是引用，代表父类的数据空间，当子父类中出现同名现象时，使用 super来进行区分

6、子类在重写父类中的方法的时候，子类中的方法的权限必须大于等于父类中方法的权限

7、为什么用重写？

沿袭父类功能，改变功能的实现方式

Zi(String name,int age)
{
    super(name,age);// 必须写在第一行
}

8、多态的前提条件：存在继承或实现

多态的弊端：只能使用父类中定义的功能，子类特有的功能不能使用了

多态的好处：提高程序的扩展性，前期定义的功能可以被后期出现的事物使用

public static void chi(Animal animal)

Animal animal =  new Cat(); 
Animal animal = new Dog();

{
    animal.eat();
}

9、toString()方法返回的是对象的字符串表示 Person@1b8d6f7
