Activity传值得三种方式

Posted on 2015-11-09 | In 安卓基础

Hello，这是android基础的第二堂课

1、ActivityPassValueForIntent

//使用Intent启动另一个 Activity并传递数据
Intent intent = new Intent(this,TargetActivity.class);

//把要传递的数据封装到intent中
intent.putExtra("name","lisi");
//intent.putExtra("age", 23);

Person person = new Person();
person.setName("小花");
person.setMarried(true);
List<String> list = new ArrayList<String>();
list.add("eat");
list.add("sleep");
list.add("play");
person.setAihao(list);
//传递的对象必须是序列化的
intent.putExtra("person",person);

//也可以使用 Bundle传递数据得到传递过来的数据在另一个Activity中
//得到启动该Activity 的Intent对象
Intent intent = getIntent();
//从Intent对象中获取数据
String name = intent.getStringExtra("name");
int age = intent.getIntExtra("age", 20);
Person person = (Person) intent.getSerializableExtra("person");

String mingzi = person.getName();
boolean married = person.isMarried();
List<String> list = person.getAihao();

3、ActivityPassValueForResult

使用该方法可以得到被启动的acivity返回的数据
第二个参数是请求码
startActivityForResult(intent, REQUESTCODE);
当从TargetActivity返回到MainAcvitity时，才能得到返回的数据，会回调该方法，参数data接收的是返回的Intent对象
protected void onActivityResult(int requestCode, int resultCode, Intent data) {
super.onActivityResult(requestCode, resultCode, data);
    if(requestCode==REQUESTCODE&&resultCode==this.RESULT_OK)
    {
        String value = data.getStringExtra("hai");
        Toast.makeText(this, value, Toast.LENGTH_LONG).show();
    }
}    
设置返回的值 
Intent inte = new Intent();
inte.putExtra("hai", "哥们,你好");    
把数据返回
第一个参数是结果码
第二个参数是含有返回的数据的Intent对象
setResult(this.RESULT_OK, inte);
