显示意图及隐式意图

Posted on 2015-11-23 | In 安卓基础

Hello，这是android基础的第五堂课

1、显式意图—明确指明了要启动的Activity
Intent intent = new Intent(this,TargetActivity.class);
//显示意图的第二种方式
//Intent intent = new Intent();
//intent.setClass(this, TargetActivity.class);
//intent.setClassName(this, “com.qianfeng.gp10_day25_intent1.TargetActivity”);
//intent.setClassName(“com.qianfeng.gp10_day25_intent1”,”com.qianfeng.gp10_day25_intent1.TargetActivity”);

//显示意图的第三种方式
//Intent intent = new Intent();
//ComponentName com = new ComponentName(this, TargetActivity.class);
//ComponentName com = new ComponentName(this, "com.qianfeng.gp10_day25_intent1.TargetActivity");
//ComponentName com = new ComponentName("com.qianfeng.gp10_day25_intent1","com.qianfeng.gp10_day25_intent1.TargetActivity");
//intent.setComponent(com);

startActivity(intent);

2、隐式匹配的最低配置：一个action,一个category,而且category必须是android.intent.category.DEFAULT

<activity android:name=".TargetActivity">
    <intent-filter >
        <action  android:name="haha.hehe.heihei"/>
        <action  android:name="com.qianfeng.gp10_day25_intent1.TargetActivity"/>
        <category android:name="com.qianfeng.gp10"/>
        <category android:name="android.intent.category.DEFAULT"/>
    </intent-filter>
</activity>
 当有多个匹配的activity时，会列出所有匹配的，让用户选择
<activity android:name=".ThirdActivity">
    <intent-filter >
        <action android:name="haha.hehe.heihei"/>
        <category android:name="com.qianfeng.gp10"/>
        <category android:name="android.intent.category.DEFAULT"/>
    </intent-filter>

</activity>

3、隐式意图—通过Action和Category来指明要启动的Activity

Intent intent = new Intent();

intent.setAction("haha.hehe.heihei");
intent.addCategory("com.qianfeng.gp10");

startActivity(intent);//默认加入了 category:android.intent.category.DEFAULT
